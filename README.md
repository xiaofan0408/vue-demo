# Vue学习大纲

## 开发环境

- [Node.js](https://nodejs.org/zh-cn/) >= 8.9.0
- [Visual Studio Code](https://code.visualstudio.com/) (编辑器)




- [Vue-CLI](https://github.com/vuejs/vue-cli)
```bash
npm install -g @vue/cli
# or
yarn global add @vue/cli
```
